# ExpiryRx Professional MVP

A polished, mobile-first static web app for high-cost medication expiry management.

## Improvements in this release
- New ExpiryRx branding
- Professional hospital-style dashboard
- Responsive cards and metrics
- Light and dark modes
- Loading animation and toast messages
- Improved alerts, recommendations, and activity log
- Kymriah demo record
- LocalStorage persistence
- PWA support

## Publish on GitHub and Vercel
1. Extract the ZIP file.
2. Upload `index.html`, `manifest.json`, `sw.js`, and `README.md` directly to the root of your GitHub repository.
3. Commit the changes.
4. Vercel should redeploy automatically. Otherwise, open the project and choose Redeploy.

## Note
This MVP stores information in the browser only. A production deployment requires authentication, a secure backend, a database, role permissions, audit controls, backups, and validated clinical workflows.
